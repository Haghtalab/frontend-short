## Slideshow

Maak een SlideShow component in React dat gebruikt kan worden om een lijst van images weer te geven.

```
<SlideShow images={[
  'http://example.com/img1.png',
  'http://example.com/img2.png',
  'http://example.com/img3.png',
]}>
```
